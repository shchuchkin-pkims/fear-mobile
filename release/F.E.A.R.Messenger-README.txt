====================================
F.E.A.R. Messenger v0.3.1
====================================

ОПИСАНИЕ:
---------
F.E.A.R. Messenger - это кроссплатформенный мессенджер с поддержкой зашифрованных аудиозвонков.
Приложение совместимо с PC версией и обеспечивает безопасную передачу аудио в реальном времени.

УСТАНОВКА:
----------
1. Скопируйте файл F.E.A.R.Messenger-v0.3.1-release.apk на телефон
2. Разрешите установку из неизвестных источников в настройках Android
3. Откройте APK файл и следуйте инструкциям установки

ТРЕБОВАНИЯ:
-----------
- Android 7.0 (API 24) или выше
- Разрешения:
  * Микрофон - для аудиозвонков
  * Интернет - для передачи данных
  * Foreground Service - для работы в фоне при заблокированном экране
  * Wake Lock - для поддержания активности во время звонка

ОСОБЕННОСТИ:
------------
✓ Зашифрованные аудиозвонки (AES-GCM)
✓ Кодек Opus (48kHz, mono, 24kbps)
✓ Работает при заблокированном экране
✓ Низкая задержка передачи звука
✓ Совместимость с PC версией

ИСПОЛЬЗОВАНИЕ:
--------------
1. Запустите приложение
2. Введите IP адрес сервера/комнаты
3. Нажмите "Join Room" для подключения к комнате
4. Нажмите "Audio Call" для начала аудиозвонка
5. Используйте "End Call" для завершения звонка
6. "Disconnect" для выхода из комнаты

ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ:
---------------------------
- Версия: 0.3.1
- Размер: ~7.1 MB
- Sample Rate: 48000 Hz
- Каналы: Mono
- Битрейт: 24 kbps
- Frame Size: 960 samples (~20ms)
- Шифрование: AES-GCM с 12-байтовым nonce
- Протокол: UDP

ИЗВЕСТНЫЕ ПРОБЛЕМЫ:
-------------------
- При первом запуске необходимо предоставить разрешения на микрофон
- При заблокированном экране отображается уведомление "Audio Call Active"
- Групповые звонки (3+ участников) пока работают нестабильно

УСТРАНЕНИЕ НЕПОЛАДОК:
---------------------
1. Нет звука:
   - Проверьте разрешение микрофона
   - Убедитесь что громкость не на минимуме

2. Связь прерывается:
   - Проверьте стабильность Wi-Fi/мобильного интернета
   - Убедитесь что приложению разрешена работа в фоне

3. Высокая задержка:
   - Проверьте скорость интернет-соединения
   - Попробуйте переключиться на Wi-Fi

БЕЗОПАСНОСТЬ:
-------------
Все аудиопакеты шифруются с использованием AES-GCM.
Ключ шифрования генерируется при подключении к комнате.
